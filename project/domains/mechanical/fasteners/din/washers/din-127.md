# Standard Overview

DIN 127 specifies the helical spring lock washer (split ring washer, commonly called Grover washer). The single-turn coil spring washer is split and offset — when compressed flat by tightening, it stores spring energy that maintains bolt tension under vibration. However, engineering consensus (Junker 1969 vibration test results) shows limited effectiveness compared to prevailing torque or positive locking methods.

Direct GOST equivalent: GOST 6402-70. Indian equivalent: IS 3063. ISO 8738 provides approximate ISO equivalent. Unlike most DIN fastener standards, DIN 127 remains in active use in German engineering drawing practice.

# Limitations

- **Not recommended** for class 8.8+ bolt connections under vibration — Junker vibration test shows spring lock washers loosen under transverse vibration
- Do not use with property class >5.8 (GOST rule; also ISO guidance)
- Single-use: once compressed flat, replace before re-use
