# Standard Overview

DIN 916 was the German standard for cup point socket set screws. **Withdrawn** and replaced by ISO 4029. Cup point is the most common set screw globally. No GOST equivalent — ISO 4029 specified directly. See standards/iso/set_screws/4029.md for full technical content.
