# Standard Overview

GOST 11738-84 specifies hexagon socket head cap screws — the fastener type universally known as an Allen bolt or cap screw. These screws feature a cylindrical head with a recessed internal hexagon socket, driven by an Allen (hex) key. The internal drive allows installation in confined spaces where a spanner cannot reach, and enables higher torque application than cross-recessed drives.

GOST 11738-84 aligns directly with ISO 4762 and the now-withdrawn DIN 912; a screw made to any of these three standards is fully interchangeable with the others in the same nominal diameter and property class. The Indian equivalent is IS 2269. Standard property classes are 8.8, 10.9, and 12.9 in steel, and A2-70 / A4-70 / A4-80 in stainless steel.

Key applications: machine tool assemblies, gearbox housings, precision fixtures, automotive components, and any joint requiring high clamping force with a clean or recessed head profile.

# Surface Treatments

| Coating | Use case | Notes |
|---------|----------|-------|
| Black oxide + oil (code 40) | Class 12.9; general engineering | No hydrogen embrittlement risk |
| Zinc electroplating 6 µm (code 01) | Class 8.8, 10.9 | Bake mandatory for 10.9 |
| Mechanical zinc plating | Class 10.9, 12.9 | No hydrogen risk — preferred |
| Geomet / Dacromet | Class 12.9; automotive | Superior corrosion; no hydrogen risk |
| Stainless A2/A4 | Corrosive, food, pharma | Anti-seize essential on assembly |

> **⚠ SOCKET FIT AFTER PLATING:** 6 µm zinc reduces the socket bore diameter. Verify hex key fit on a plated sample before bulk acceptance.

> **⚠ CLASS 12.9 MARKING RULE:** Marking must be embossed (raised) or on side face only — never stamped into the head bearing face.

> **⚠ STAINLESS ANTI-GALLING:** Apply anti-seize (Molykote G-Rapid or equivalent). Tighten slowly < 1 rev/s. Do not impact-drive stainless socket screws.

# Marking Requirements

- Property class on head top face or cylindrical side face
- Class 12.9: embossed or side face only — no indented marking on top face
- Manufacturer code mandatory for class 8.8 and above
- Stainless: "A2-70", "A4-70", "A4-80"
