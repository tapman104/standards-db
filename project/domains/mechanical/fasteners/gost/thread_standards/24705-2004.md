# Standard Overview

GOST 24705-2004 provides the calculated basic dimensions for metric threads — pitch diameter, minor diameter, and major diameter for both external (bolt) and internal (nut) threads. These values are derived from the profile geometry in GOST 9150-2002 and the pitch combinations in GOST 8724-2002.

Consistent with ISO 724 and IS 4218-3. Together with GOST 9150-2002 and GOST 8724-2002, this standard completes the metric thread dimensional system, ensuring full interchangeability between Russian and international metric threaded fasteners.

# Thread Dimension Relationships

From fundamental thread geometry (derivable from profile and pitch):
- **Pitch diameter (d2):** d − 0.6495 × P
- **Minor diameter (d3/D1):** d − 1.2269 × P (external); d − 1.0825 × P (internal)
- Where d = nominal (major) diameter, P = pitch

These relationships are standardised, public mathematical facts derivable from first principles.

# Interchangeability Statement

Any metric fastener manufactured to GOST 9150-2002 + GOST 8724-2002 + GOST 24705-2004 is fully dimensionally interchangeable with metric fasteners manufactured to ISO 68-1 + ISO 261 + ISO 724 or to IS 4218 Parts 1–3.
