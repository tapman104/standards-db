# Standard Overview

GOST 9150-2002 establishes the fundamental 60-degree thread profile for metric threads used in Russian engineering. This profile defines the basic geometry — thread angle, root radius, crest form — that all metric threaded fasteners in the GOST system conform to.

GOST 9150-2002 is dimensionally identical to ISO 68-1, which defines the same 60° metric thread profile internationally. Any metric fastener manufactured to GOST thread profile specifications is geometrically interchangeable with ISO/IS/DIN metric threaded fasteners.

# Thread Profile Facts

| Parameter | Value |
|-----------|-------|
| Thread angle | 60° (included angle) |
| Profile | ISO metric — V-thread with rounded root |
| Root | Radiused (reduces stress concentration) |
| Crest | Flat or slightly rounded |

# Compatibility Note

Full dimensional interchangeability between GOST, ISO, DIN, and IS metric fasteners is achieved through the harmonisation of these three thread standards (GOST 9150, GOST 8724, GOST 24705). See companion standards.
