# Standard Overview

GOST 5927-70 specifies precision-grade (grade A) hexagon nuts of style 1, manufactured to tighter dimensional tolerances compared to GOST 5915-70 (grade B). Grade A nuts are limited to diameters up to M16 and are used where closer fit and higher precision are required: instrumentation, precision machinery, metrology equipment, and quality-critical assemblies.

Dimensionally equivalent to ISO 4032 grade A and the Indian IS 1364 Part 3. Strength classes 5 through 12 are available; the nut property class must match or exceed the paired bolt property class.

# Surface Treatments

Same as GOST 5915-70. Grade A nuts for precision applications are often supplied bare or phosphate + oil to avoid coating-induced dimensional variation.

# Marking Requirements

Same location as GOST 5915-70 (side flat). Property class + manufacturer code for class 8+.

# Reliability Notes

- HIGH confidence: confirmed across multiple sources
- Pair with grade A bolts (GOST 7817-80) for fully precision assemblies
- For M16+ nuts, use GOST 5915-70 (grade B) as grade A is not available above M16
