# Standard Overview

GOST 50592-93 specifies prevailing torque lock nuts that use a permanently deformed non-metallic (nylon/polyamide) insert to create friction against the mating bolt thread, resisting loosening under vibration and dynamic loads. The nylon insert is undersized relative to the bolt thread and does not cut into it, allowing limited reuse.

These are among the most widely used locking fasteners in general machinery, automotive, and consumer product assembly. Low-form variants correspond to DIN 985 / ISO 7040. The Indian standard IS 7002 covers prevailing torque hexagon nuts of this type.

**Key limitation:** The nylon insert limits service temperature to approximately 120°C continuous. For higher temperatures, all-metal lock nuts (ISO 7042 / DIN 980) must be substituted.

# Materials

| Component | Material |
|-----------|---------|
| Nut body | Carbon steel (classes 5–10) or stainless A2/A4 |
| Nylon insert | PA6 (polyamide 6) or PA66 (polyamide 66) |

PA66 continuous service limit ~130°C; PA6 ~120°C. Both fail rapidly above these limits. Do not use in environments with strong acids, solvents, or hydraulic oils at elevated temperature.

| Treatment | Compatibility | Notes |

| Hot-dip galvanizing | NO | 450°C bath destroys nylon insert — PROHIBITED |

> **⚠ CRITICAL:** Hot-dip galvanizing (450°C) will melt and destroy the polyamide insert. HDG is absolutely prohibited for nylon insert lock nuts.

# Marking Requirements

- Property class number on one hexagon flat face: "5", "6", "8", "10"
- Stainless: "A2" or "A4" with strength code if applicable
- Nylon insert colour is manufacturer-specific — NOT a standardised property class indicator
- Do not rely on insert colour alone to identify property class

# Testing Requirements

ISO 2320 prevailing torque test (5-cycle protocol):

| Thread | Min torque — 1st on-cycle (Nm) | Min torque — 5th on-cycle (Nm) |
|--------|-------------------------------|-------------------------------|
| M6 | 1.8 | 1.2 |
| M8 | 3.0 | 2.0 |
| M10 | 5.0 | 3.4 |
| M12 | 8.0 | 5.4 |
| M16 | 15 | 10 |
| M20 | 25 | 17 |
| M24 | 36 | 24 |

Values shown are for class 6, coarse thread, dry condition. Class 8 and 10 values are ~30–40% higher.

Additional tests:
- Proof load: no thread stripping or fracture
- Hardness: within property class limits
- Off-torque: must remain manually removable (max prevailing torque not exceeded)

# Reusability

Maximum **5 assembly cycles** per ISO 2320. This is not a recommendation — it is the test basis.

**Mandatory discard regardless of cycle count:**
- Measured prevailing torque falls below ISO 2320 minimum for the size/class
- Visible nylon damage (cracking, deformation beyond thread profile)
- Any heat exposure above 120°C — insert no longer functions reliably on reuse
- Any contact with strong solvents, concentrated acids, or hydraulic oil at elevated temperature

# Reliability Notes

- HIGH confidence: GOST 50592-93 = ISO 7040 = DIN 985 confirmed (FCH sourcing network + multiple)
- Low form only — for high form see GOST 11872-89 (= ISO 7041 / DIN 982)
- IS 7002 covers both low and high form nylon insert nuts under a single standard
- Temperature limit of 120°C is firm — use ISO 7042 / DIN 980 for elevated temperature service
