# Standard Overview

GOST 15589-70 covers lower-precision (grade C) hexagon head bolts intended for general construction and structural applications where close tolerances are unnecessary. The wider dimensional tolerances (grade C) reduce manufacturing cost, making these bolts economical for bulk civil and structural applications such as timber connections, general steelwork, bolted gusset plates, and non-critical machinery frames.

Property classes are limited to 4.6, 5.6, and 5.8 — high-strength classes (8.8, 10.9) are not applicable to grade C bolts. The equivalent international standard is ISO 4016; the Indian equivalent is IS 1363 Part 1, commonly called "black bolts" in the Indian construction market.

# Dimensions

| Parameter | Value |
|-----------|-------|
| Thread range | M5 to M64 |
| Thread tolerance | 8g (wider than grade A/B 6g) |
| Product grade | C |
| Shank | Partially threaded |
| WAF | Wider dimensional tolerance band than grade A/B |

Grade C bolts have wider tolerances on all dimensions including head height, WAF, and shank diameter. They are intentionally designed for use in clearance holes with significant play.

# Mechanical Properties

| Property Class | Min Tensile (MPa) | Min Yield (MPa) | Hardness (HV) |
|----------------|-------------------|-----------------|---------------|
| 4.6 | 400 | 240 | 120–220 |
| 5.6 | 500 | 300 | 155–220 |
| 5.8 | 500 | 400 | 155–220 |

Grade C bolts are not available in property class 8.8 or higher.

# Materials

| Property Class | Typical Steel |
|----------------|--------------|
| 4.6 | Low carbon — Steel 10, Steel 20 |
| 5.6/5.8 | Low-medium carbon — Steel 20, Steel 35 |

# Surface Treatments

Grade C bolts are most commonly supplied in bare/black oxide finish. Hot-dip galvanizing (GOST 9.307-89 / ISO 10684) is standard for outdoor structural applications.

# Marking Requirements

- Property class on head face or side flat for M5 and above
- Manufacturer code when specified
- Grade C is not marked on the fastener — it is identified through the GOST number in the designation

# Testing Requirements

| Test | Criterion |
|------|-----------|
| Tensile strength | Min Rm per property class |
| Hardness | Within HV range per class |
| Dimensional | Per GOST 15589-70 grade C tolerance field |

# Reliability Notes

- HIGH confidence for ISO 4016 and DIN 601 equivalency
- MEDIUM confidence for IS 1363-1 — WAF dimensions should be verified before production substitution
- Grade C bolts are NOT suitable for structural steel friction-grip (HSFG) connections
- Do not specify with property classes above 5.8
