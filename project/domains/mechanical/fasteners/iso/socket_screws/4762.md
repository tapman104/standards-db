# Standard Overview

ISO 4762 specifies socket head cap screws with hexagonal (Allen) drive socket. This is the current primary international reference, replacing the withdrawn DIN 912. The socket drive allows installation in confined spaces inaccessible to spanner tools, and the high-profile cylindrical head provides a large clamping area. Available in property classes 8.8, 10.9, and 12.9 as well as stainless.

Direct GOST equivalent: GOST 11738-84. Indian equivalent: IS 2269. DIN 912 was the German predecessor — withdrawn and replaced by ISO 4762.

# Anti-Galling (Stainless)

Apply never-seize or thread anti-galling compound to stainless A2/A4 socket screws before installation. Stainless-on-stainless galling risk is high.
