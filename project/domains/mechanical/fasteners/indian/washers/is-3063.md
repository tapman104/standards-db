# Standard Overview

IS 3063 specifies helical spring lock washers (Grover washers) for metric threaded fasteners. Equivalent to GOST 6402-70 and DIN 127. Hardness 41.5–49.5 HRC. Single-use.

**Limitation:** Not recommended for class 8.8+ under vibration loading — Junker test shows limited effectiveness. Use with property class ≤5.8 only per GOST guidance.
