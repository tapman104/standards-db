# Standard Overview

DIN 931 was the German standard for partial-thread hexagon head bolts. It has been **withdrawn** and replaced by ISO 4014. Dimensions are largely identical, with minor WAF (width across flats) differences at M10, M12, M14, and M22 where the ISO 4014 standardised on smaller WAF values for international harmonisation.

Legacy components stamped with DIN 931 remain in service. Replacement orders should specify ISO 4014; spare parts suppliers still commonly reference DIN 931 for this reason.

# WAF Differences DIN 931 vs. ISO 4014

| Size | DIN 931 WAF (mm) | ISO 4014 WAF (mm) | Note |
|------|-----------------|------------------|------|
| M10 | 17 | 16 | DIN larger |
| M12 | 19 | 18 | DIN larger |
| M14 | 22 | 21 | DIN larger |
| M22 | 32 | 34 | ISO larger |

> **FLAG:** Verify socket/spanner compatibility before substituting ISO 4014 for DIN 931 at these sizes.

# Withdrawal Note

DIN 931 is **withdrawn**. New designs must specify ISO 4014. Existing plant maintenance drawings referencing DIN 931 may be ordered against ISO 4014 with WAF verification at the four flagged sizes.
