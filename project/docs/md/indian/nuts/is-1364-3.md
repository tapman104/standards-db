# Standard Overview

IS 1364 Part 3 specifies hexagon nuts style 1 (standard height) for metric threads. **BIS has declared IS 1364-3 as identical to ISO 4032** — this is a formal harmonisation declaration meaning IS 1364-3 is not an adaptation but a direct adoption. Full dimensional and mechanical property equivalence with ISO 4032.

This formal identity declaration makes IS 1364-3 nuts fully interchangeable with ISO 4032 nuts for engineering specification purposes.
