# Standard Overview

IS 2016 specifies plain flat washers for metric threaded fasteners. Covers both chamfered and non-chamfered variants, corresponding to ISO 7090 (chamfered) and ISO 7089 (plain). Direct GOST equivalent: GOST 11371-78.
