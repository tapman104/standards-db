# Standard Overview

GOST 11872-89 specifies the high-form nylon insert lock nut — dimensionally taller than the low-form variant (GOST 50592-93) for the same nominal thread diameter. The additional height provides greater thread engagement depth with the bolt, increasing resistance to strip-out under high axial load and providing marginally higher prevailing torque.

High-form variants correspond to DIN 982 / ISO 7041. The same temperature limitation (120°C maximum continuous) and reusability rules (5 cycle maximum) apply as for the low-form version. The Indian standard IS 7002 covers both forms.

# Mechanical Properties

Same as GOST 50592-93. Identical property class range (5, 6, 8, 10; stainless A2, A4).

# Materials

Same as GOST 50592-93. Nut body: carbon steel or stainless. Insert: PA6 or PA66.

# Surface Treatments

Same restrictions as GOST 50592-93. Hot-dip galvanizing **prohibited** — destroys nylon insert.

# Marking Requirements

Same as GOST 50592-93. Property class on side flat; stainless designation on side flat.

# Testing Requirements

Same ISO 2320 prevailing torque protocol as GOST 50592-93 (5-cycle test). See 50592-93.md for full torque table.
