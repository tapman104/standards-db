# Standard Overview

GOST 8724-2002 specifies the preferred diameter-pitch combinations for coarse and fine metric threads, matching ISO 261 (general metric screw thread series) and ISO 262 (selected sizes). The standard defines which pitch values are standard (coarse) and which are fine for each nominal diameter.

Fully harmonised with ISO 261/262 and IS 4218-2. Coarse thread is the default for all general-purpose fasteners; fine thread must be stated explicitly in the designation string.
