# Standard Overview

GOST 7805-70 defines fully threaded hexagon head bolts where the metric thread extends the entire length of the shank from just below the head. This design is preferred when bolts thread directly into tapped holes (no nut required) or when maximum thread engagement is needed. The full-thread shank carries no shear load; clamping force is developed entirely through thread engagement. Available across a wide diameter range (M1.6 to M64) with property classes from 4.6 up to 12.9 in steel, and A2-70 / A4-70 in stainless steel.

The standard aligns directly with ISO 4017 and its German counterpart DIN 933. The Indian equivalent is IS 1364 Part 2. Key applications include mechanical assemblies, agricultural equipment, construction fixings, and food-industry equipment when manufactured in stainless steel.

Same coating code table as GOST 7798-70. See Category 1 Section 9 of the master research document.

**Class 12.9 specific:** Embossed or side-face marking only — never indented marking on head bearing face.

# Marking Requirements

- Property class marked on head face or hexagon side flat
- Manufacturer code mandatory for class 8.8, 10.9, 12.9
- **Class 12.9:** marking must be embossed (raised) or on side face — never stamped/indented on top face (stress concentration risk at socket lip)
- Stainless marked "A2-70" or "A4-70" (combined alloy + strength designation)

# Testing Requirements

Same test matrix as GOST 7798-70. Additional for class 12.9:
- Surface hardness check: max surface hardness ≤ 30 HV above measured core hardness
- Stress corrosion (delayed fracture) — per ISO 15330 where specified

# Reliability Notes

- HIGH confidence: GOST 7805-70 = ISO 4017 = DIN 933 confirmed across 3+ sources
- Full thread range wider than GOST 7798-70 (starts M1.6 vs M6)
- Do not use class 12.9 with acid-bath zinc plating — hydrogen embrittlement risk
- DIN 933 is the legacy designation; ISO 4017 is the current preferred reference
