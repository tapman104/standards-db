# Standard Overview

GOST 24379.1-80 specifies the dimensional requirements for seven distinct types of foundation bolts, including straight, bent (L-bolt and J-bolt), plate anchor, removable sleeve, double-end threaded, and composite anchor varieties. Bolt sizes span M12 to M125 with lengths extending to 5 metres for deep foundation applications.

These anchors are embedded in concrete before or after pouring, depending on type, and are used to mount industrial equipment, turbines, pumps, generators, structural columns, and machinery bases. The Indian equivalent IS 5624 covers similar shank form variants (Types A–G) with dimensional coverage up to M48. Both standards permit hot-dip galvanising for corrosion protection.

# Dimensions

| Parameter | Value |
|-----------|-------|
| Thread range | M12 to M125 |
| Nominal lengths | 150 mm to 5000 mm |
| Thread type | Metric coarse |
| Thread tolerance | 6g |
| Matched nut | GOST 5915-70 or GOST 5927-70 |
| Matched washer | GOST 11371-78 or DIN 797 (heavy washer for anchor applications) |

**Foundation bolt types:**

| Type | Form | Installation method |
|------|------|-------------------|
| 1 | Straight shank | Grouted into pre-drilled hole or cast-in |
| 2 | L-bend (90°) | Cast-in with bend as anchor |
| 3 | J-bend (hook) | Cast-in with hook as anchor |
| 4 | Plate anchor | Cast-in with end plate for pull-out resistance |
| 5 | Removable sleeve | Sleeve cast-in; bolt inserted after concrete cures |
| 6 | Double-end threaded | Threaded both ends; nut below slab + nut above |
| 7 | Composite | Special anchor head cast-in; threaded rod installed later |

# Mechanical Properties

Same as GOST 24379.0-80. Typically class 4.6 or 5.6. Class 8.8 for heavy machinery.

# Materials

Same as GOST 24379.0-80. Low carbon steel (Steel 20) for standard anchors; medium carbon alloy steel (Steel 40X) for class 8.8 seismic applications.

# Surface Treatments

| Zone | Treatment |
|------|-----------|
| Embedded in concrete | Hot-dip galvanizing (GOST 9.307) or bare (alkaline passivation by concrete) |
| Exposed thread above concrete | Hot-dip galvanizing or heavy zinc electroplating |
| Aggressive soil / chemical | Epoxy coating entire length |

Minimum HDG thickness on foundation bolts per GOST 9.307-89:
- M12–M24: ≥55 µm
- M27–M48: ≥70 µm
- >M48: ≥85 µm

# Marking Requirements

- Type designation (1–7) + thread size + length + property class on a durable tag attached to each bolt or bundle
- Embossed or painted marking on the exposed end for post-installation identification
- GOST 24379.1 in the purchase order documentation

# Testing Requirements

| Test | Notes |
|------|-------|
| Tensile strength | On representative sample from lot |
| Bend test | Types 2 and 3: 180° bend test at bend radius — no cracking |
| Dimensional | Thread gauge (6g) + length + shank diameter |
| Visual / surface | No laps, seams, or corrosion before installation |

# Reliability Notes

- HIGH confidence: DIN 529 and IS 5624 equivalence confirmed for main types
- IS 5624 max thread size M48; for larger diameters only GOST and DIN 529 apply
- Removable sleeve bolts (Type 5): sleeve and nut are reusable; inspect bolt rod for corrosion/bending before reinstallation
- Foundation bolts are permanent once grouted — ensure correct size, type, and position before concrete pour
