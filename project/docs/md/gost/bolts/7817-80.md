# Standard Overview

GOST 7817-80 specifies precision-grade (grade A) hexagon head bolts with a partially threaded shank, covering diameters up to M24. Product grade A imposes tighter manufacturing tolerances on all critical dimensions compared to grade B (GOST 7798-70). This makes these bolts suitable for precision engineering, instrumentation, metrology fixtures, machine tool assemblies, and applications where close fits and minimal play are required.

The standard occupies the same functional space as ISO 4014 product grade A. Grade A nuts (GOST 5927-70) and plain washers precision grade A (GOST 11371-78) are the matched components for complete precision assemblies.

**Scope difference from GOST 7798-70:**
- GOST 7798-70 = Grade B, M6–M48
- GOST 7817-80 = Grade A, M1.6–M24

# Surface Treatments

Same coating code table as GOST 7798-70. Precision grade A bolts are frequently supplied bare (code 00) or phosphate + oil (code 40) when used in precision instruments, as plating build-up can affect fits.

# Marking Requirements

Same as GOST 7798-70. Property class and manufacturer code on head face or side flat. Grade marking is confirmed through diameter range and tolerance specification in the designation string — "Grade A" is implied when specifying GOST 7817-80.

# Testing Requirements

Same test matrix as GOST 7798-70, with additional dimensional verification for tighter tolerance fields applicable to grade A.

# Reliability Notes

- HIGH confidence: grade A mapping confirmed
- Grade A scope limited to ≤M24 — for M24+ partial thread bolts use GOST 7798-70 (grade B)
- Use GOST 5927-70 (grade A nut) as the matched nut for this bolt
- Grade A plain washer: GOST 11371-78 accuracy class A
