# Standard Overview

GOST 24379.0-80 establishes the general technical requirements applicable to all types of foundation (anchor) bolts used to secure equipment, machinery, and structural columns to concrete or masonry foundations. It covers material grades, mechanical properties, dimensional tolerances, surface treatment, marking, and acceptance rules that apply across the entire GOST foundation bolt series.

This standard functions as the umbrella specification under which the specific bolt type dimensions are detailed in the companion standard GOST 24379.1-80. Any foundation bolt manufactured under the GOST 24379 series must comply with the general requirements of this document.

# Dimensions

Dimensional requirements are in GOST 24379.1-80. This standard establishes:
- Tolerance fields applicable to all foundation bolt types
- Thread tolerances (6g for metric threads)
- Straightness requirements for long shank bolts
- Bend radius requirements for L- and J-type bolts

# Mechanical Properties

| Property Class | Min Tensile (MPa) | Min Yield (MPa) |
|----------------|-------------------|-----------------|
| 4.6 | 400 | 240 |
| 5.6 | 500 | 300 |
| 8.8 | 800 | 640 |

Foundation bolts are most commonly supplied in class 4.6 or 5.6. Class 8.8 is used for heavy machinery anchoring and seismic-zone installations.

# Materials

| Grade | Typical Steel | Notes |
|-------|--------------|-------|
| Low carbon | Steel 10, Steel 20 | Standard for classes 4.6 and 5.6 |
| Medium carbon | Steel 35, Steel 40X | For class 8.8 anchors |

# Surface Treatments

| Treatment | Application |
|-----------|-------------|
| Hot-dip galvanizing (GOST 9.307) | Standard for outdoor and embedded anchors |
| Zinc electroplating | Above-grade exposed thread section only |
| Bare (embedded portion) | Acceptable in dry concrete environments |
| Epoxy coating | Aggressive soil / chemical environments |

Hot-dip galvanizing is the standard corrosion protection for foundation bolts embedded in concrete. The embedded section relies on alkaline concrete environment for passivation.

# Marking Requirements

- Property class on exposed thread end or a tag attached to the bolt
- Manufacturer code for class 8.8
- GOST 24379 identification
- Type designation (bolt type A–G) per GOST 24379.1-80

# Testing Requirements

| Test | Notes |
|------|-------|
| Tensile strength | Min per property class |
| Bend test (L/J types) | No cracking at bend radius |
| Dimensional | Per GOST 24379.1-80 tolerance tables |
| Thread gauge check | 6g go/no-go gauge |

# Designation Examples

**GOST format:**
```
Foundation bolt M24 × 500 Type 1 — 4.6 — 13 GOST 24379.1-80
(Type 1 = straight shank; length 500 mm; class 4.6; hot-dip galvanized)
```
