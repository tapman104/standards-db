# Standard Overview

GOST 1759.5-87 specifies the mechanical and physical property requirements for metric hexagon nuts across property classes 4 through 12. It defines proof load stress, hardness, and material requirements for each class, and establishes the compatibility rules between bolt and nut property classes.

Equivalent to ISO 898-2. The Indian equivalent is IS 1367 Part 6.

# Property Classes and Pairing Rules

Nut property class must be selected to match the bolt to avoid thread stripping at the nut:

| Nut Class | Min Proof Load Stress (MPa) | Hardness HV (min–max) | Compatible Bolt Classes |
|-----------|-----------------------------|-----------------------|------------------------|
| 4 | 400 | 130–302 | 3.6, 4.6, 4.8 |
| 5 | 500 | 130–302 | 3.6, 4.6, 4.8, 5.6, 5.8 |
| 6 | 600 | 150–302 | 5.6, 5.8, 6.8 |
| 8 | 800 | 200–353 | 6.8, 8.8 |
| 10 | 1000 | 272–353 | 10.9 |
| 12 | 1200 | 295–353 | 12.9 |

# Materials

| Nut Class | Steel | Notes |
|-----------|-------|-------|
| 4, 5, 6 | Low-medium carbon steel | No heat treatment required |
| 8 | Medium carbon or carbon-boron | May be heat treated |
| 10, 12 | Alloy steel | Heat treated |

# Testing Methods

| Test | Method |
|------|--------|
| Proof load (axial) | Thread mandrel loaded to proof load — no stripping, no fracture |
| Hardness (Vickers) | ISO 6507-1 on bearing face — min/max HV per class |
| Dimensional | Thread 6H gauge check + WAF + height |
