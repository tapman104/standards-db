# Standard Overview

GOST 6402-70 specifies single-coil split spring washers (colloquially known as Grover washers or shake-proof washers) used to resist spontaneous loosening of threaded connections under vibration and dynamic loads. The split ends bite into the nut and bearing surface when reverse rotation is attempted, creating frictional resistance to loosening.

**Critical limitation:** Spring washers are only effective and only permitted with bolts of property class 5.8 or lower. With class 8.8 or higher bolts, the hard bolt bearing surface overrides the spring washer effect — the washer simply embeds and flattens without providing adequate locking. Four cross-section types (H, T, OT, L) cover light to extra-heavy duty applications.

The direct German equivalent is DIN 127 (Form B for GOST Version 1, Form A for GOST Version 2). The Indian equivalent is IS 3063.

# Surface Treatments

| Treatment | Notes |
|-----------|-------|
| Black oxide + oil | Standard — adequate for indoor use |
| Zinc electroplating | Light coating only — heavy plating reduces spring action |
| Stainless | No coating needed |

**Avoid thick coatings:** Excessive coating on spring washers reduces the cutting action of the split ends and the effective spring height. Zinc plating ≤6 µm is acceptable.

# Reusability

**Single use only.** When compressed during assembly, the split ends lose their spring-back energy. Reused spring washers provide negligible locking action. Always replace with new washers at each assembly.

# Reliability Notes

- HIGH confidence: DIN 127 and IS 3063 equivalency confirmed
- **Do NOT use with class 8.8+ bolts** — the locking mechanism does not function reliably
- For class 8.8+ joints: use nylon insert lock nuts (GOST 50592-93) or all-metal lock nuts (ISO 7042)
- Single-use component — never reuse a spring washer
