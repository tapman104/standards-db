# Standard Overview

ISO 4029 specifies socket set screws with cup point — a concave (cupped) circular rim tip. The cup rim bites into the shaft surface, providing medium-to-high holding force with moderate surface damage. It is the most widely used set screw point type globally, offering a balance between holding force, ease of removal, and shaft reusability.

**No confirmed GOST number** — Russian drawings reference ISO 4029 directly. German predecessor: DIN 916 (withdrawn). Indian: IS 6094-4.
