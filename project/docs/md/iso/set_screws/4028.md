# Standard Overview

ISO 4028 specifies socket set screws with dog point (cylindrical or half-cylindrical protrusion at tip). The dog engages a machined recess or cross-hole in the shaft, providing the highest holding force with zero shaft surface damage. Requires pre-machined shaft recess matching the dog diameter.

Direct GOST equivalent: GOST 11075-93. German predecessor: DIN 915 (withdrawn). Indian: IS 6094-3.

# Dog Point Types

| Type | Description | Use |
|------|-------------|-----|
| Full dog | Cylindrical protrusion = full thread minor diameter | Heavy-duty; maximum holding |
| Half dog | Cylindrical protrusion = half thread minor diameter | Fine-pitch screws; tighter tolerance fit |

# Shaft Recess Requirement

The mating shaft or hub must have a machined recess (drilled flat-bottom hole or through-hole) matching the dog point diameter. Without this recess, dog point set screws provide no benefit over flat point.

# Reliability Notes

- DIN 915 withdrawn; ISO 4028 is current reference
- Best reusability of all set screw types — shaft recess is not damaged
- Half-dog rule: for pitches ≤ 1.25 mm, half dog is standard
