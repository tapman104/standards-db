# Standard Overview

ISO 4027 specifies socket set screws with cone point. The sharp cone tip bites into the shaft surface, providing high holding force against both axial and rotational movement. This high surface penetration results in shaft damage and limits reuse. Used in permanent or semi-permanent assemblies where maximum holding force is required.

**No confirmed GOST number** — Russian and Soviet technical drawings reference ISO 4027 directly. German predecessor: DIN 914 (withdrawn). Indian: IS 6094-2.
