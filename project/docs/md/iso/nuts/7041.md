# Standard Overview

ISO 7041 specifies the high-form nylon insert lock nut — style 2 (taller than ISO 7040 style 1 for the same thread diameter). The increased nut height provides greater thread engagement and marginally higher proof load capacity, making it preferable in high-load or high-vibration applications where maximum locking retention is needed.

Direct GOST equivalent: GOST 11872-89. German predecessor: DIN 982 (withdrawn, replaced by ISO 7041). Indian: IS 7002.
