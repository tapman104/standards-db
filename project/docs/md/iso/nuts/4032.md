# Standard Overview

ISO 4032 specifies the standard hexagon nut, style 1 (standard height), in product grades A and B. It is the primary nut standard used with ISO 4014 (partial thread) and ISO 4017 (full thread) bolts. Style 1 nuts have chamfered bearing face on both sides.

Grade A (precision, M1–M16), Grade B (general, M1–M64). Direct GOST equivalents: GOST 5927-70 (grade A) and GOST 5915-70 (grade B). Indian equivalent IS 1364-3 is declared identical to ISO 4032 by BIS.

# Mechanical Properties

Per ISO 898-2 / GOST 1759.5-87. Property classes 5, 6, 8, 10, 12 and stainless A2/A4.

# Pairing Rules

Match nut class to bolt class: class 8 nut with class 8.8 bolt; class 10 nut with class 10.9 bolt; class 12 nut with class 12.9 bolt.

# Surface Treatments

Per ISO 4042 (electroplating) and ISO 10684 (HDG). HDG requires tapping of thread after galvanizing — oversize thread tapped to 6AZ tolerance.

# Reliability Notes

- IS 1364-3 declared identical to ISO 4032 — no qualification difference
- DIN 934 withdrawn; ISO 4032 is current reference
- Grade A limited to M1–M16; use grade B for larger sizes
