# Standard Overview

ISO 7042 specifies the all-metal prevailing torque hexagon nut, high form. Unlike nylon insert lock nuts, ISO 7042 has no polymer component — the locking action is achieved by a deformed (oval or distorted) top section of the nut that bites into bolt threads on engagement. This makes it suitable for elevated temperatures where nylon inserts would fail.

Also called Stover nut or oval lock nut. German predecessor: DIN 980 (withdrawn, replaced by ISO 7042). No confirmed GOST equivalent for all-metal type — Russian drawings specify ISO 7042 directly. No Indian IS equivalent; specify ISO 7042 directly.

# Mechanical Properties

Property classes 5, 6, 8, 10, 12; stainless A2/A4. Class 12 all-metal lock nut available (not possible with nyloc — nylon insert proof load is the limiting factor).

# Reusability

Maximum 5 cycles (same as nyloc). The deformed top section loses locking force with each removal. Discard if prevailing torque falls below ISO 2320 minimum.

# Reliability Notes

- DIN 980 withdrawn; ISO 7042 is current reference
- Use ISO 7042 instead of ISO 7040/7041 above 120°C or when HDG is required
- All-metal locking action is generally less secure than nyloc at low temperatures
- Distorted top section must be discarded after 5 cycles
