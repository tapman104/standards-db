## SECTION A: SOCKET HEAD CAP SCREWS

---

### GOST 11738-84
**Title:** Hexagon Socket Head Cap Screw (Allen Bolt)
**Category:** Socket screw
**Also known as:** Allen bolt, cap screw, socket bolt, SHCS

**Cross-References:**
| Standard | Body | Confidence | Notes |
|----------|------|------------|-------|
| ISO 4762 | ISO | ✅ HIGH | Direct equivalent — dimensionally identical |
| DIN 912 | DIN | ✅ HIGH | Direct equivalent — withdrawn, replaced by ISO 4762 |
| IS 2269 | BIS | ✅ HIGH | Indian socket head cap screw standard |
| ASME B18.3 | ASME | 🟡 MEDIUM | US inch-series equivalent (different thread system) |

**⚠ KEY FACT:** DIN 912 and ISO 4762 are dimensionally identical for sizes M3–M64.
DIN 912 has been formally withdrawn and replaced by ISO 4762.
Fine thread variant: ISO 12474 (not DIN 912).

**Dimension facts:**
- Thread range: M1.6 to M64 (ISO 4762 scope)
- Drive: internal hexagon (Allen/hex key)
- Head: cylindrical
- Thread: fully threaded up to head (or partial for longer lengths depending on diameter)
- Thread tolerance: 6g
- Property classes: 8.8, 10.9, 12.9 (steel); A2-70, A4-70, A4-80 (stainless)
- Material: steel 45H, alloy steel (12.9), stainless A2/A4

**Torque reference (dry thread, indicative):**
| Size | Class 8.8 | Class 10.9 | Class 12.9 |
|------|-----------|-----------|-----------|
| M6 | 10 Nm | 14 Nm | 17 Nm |
| M8 | 25 Nm | 35 Nm | 41 Nm |
| M10 | 49 Nm | 69 Nm | 83 Nm |
| M12 | 84 Nm | 120 Nm | 144 Nm |

**Original Summary:**
GOST 11738-84 specifies hexagon socket head cap screws — the fastener type universally
known as an Allen bolt or cap screw. These screws feature a cylindrical head with a
recessed internal hexagon socket, driven by an Allen (hex) key. The internal drive
allows installation in confined spaces where a spanner cannot reach, and enables
higher torque application than cross-recessed drives. GOST 11738-84 aligns directly
with ISO 4762 and the now-withdrawn DIN 912; a screw made to any of these three
standards is fully interchangeable with the others in the same nominal diameter and
property class. The Indian equivalent is IS 2269. Standard property classes are 8.8,
10.9, and 12.9 in steel, and A2-70 / A4-70 in stainless steel. Key applications include
machine tool assemblies, gearbox housings, precision fixtures, automotive components,
and any joint requiring high clamping force with a clean or recessed head profile.

**Data sources consulted:** eurolinkfss.com, aimsindustrial.com.au, iso.org, boltbase.com

### ISO 7380 (Button Head Socket Screw) — companion standard
**Title:** Hexagon Socket Button Head Screw
**GOST equivalent:** No direct GOST number — specified by ISO 7380 directly in Russian engineering

**Cross-References:**
| Standard | Body | Notes |
|----------|------|-------|
| ISO 7380-1 | ISO | Button head, coarse thread |
| ISO 7380-2 | ISO | Button head with collar |
| DIN 7380 | DIN | German equivalent |

**Original Summary:**
ISO 7380 defines low-profile button head socket screws as a companion to GOST 11738-84/
ISO 4762 cap screws. The rounded button head provides a lower profile and cleaner
appearance but at approximately 30–40% lower torque capacity than the standard cylindrical
cap head. Used where clearance is limited or where aesthetics matter more than maximum
clamping force. No equivalent GOST number exists; Russian drawings typically call these
out directly as ISO 7380. Indian purchasers source these to IS 2269 (which covers the
general socket screw category) or import directly.

## SECTION B: SET SCREWS (GRUB SCREWS)

**Important note:** Set screws are headless, fully threaded screws used to lock components
onto shafts or locate parts. The tip geometry determines holding power and application.
All four GOST set screw standards use the same hexagon socket drive.

---

### GOST 11074-93
**Title:** Hexagon Socket Set Screw — Flat Point
**Category:** Set screw

**Cross-References:**
| Standard | Body | Confidence | Notes |
|----------|------|------------|-------|
| ISO 4026 | ISO | ✅ HIGH | Direct equivalent — flat point |
| DIN 913 | DIN | ✅ HIGH | Direct equivalent (DIN withdrawn, replaced by ISO 4026) |
| IS 6094-1 | BIS | 🟡 MEDIUM | Indian set screw standard Part 1 |

**Dimension facts:**
- Thread range: M1.6 to M24 (ISO 4026 scope)
- Drive: hexagon socket
- Point: flat (blunt end — large contact area, low point pressure)
- Thread tolerance: 6g
- Material: steel 45H; stainless A2, A4
- Hardness: 33–39 HRC (steel)

**Point behaviour:** Flat point distributes load over larger area. Recommended for use
against hardened surfaces or where surface marking must be minimised.

**Original Summary:**
GOST 11074-93 specifies headless set screws with a flat (blunt) end and an internal
hexagon socket drive. These screws are threaded along their entire length and are
used to secure machine components where conventional bolt-and-nut assembly is
impractical. The flat point minimises surface damage on the mating component and
provides the largest contact area of all set screw point types, making it suitable
for locking parts against hardened shafts or precision surfaces. It corresponds
directly to ISO 4026 and the withdrawn DIN 913. Applications include instrument
housings, encoder rings, mechanical stops, and adjustment mechanisms in precision
machinery.

**Data sources consulted:** dinmark.com.ua, iso.org/standard/36431, eurolinkfss.com

---

### GOST 11075-93
**Title:** Hexagon Socket Set Screw — Dog Point (Trunnion/Pin Point)
**Category:** Set screw

**Cross-References:**
| Standard | Body | Confidence | Notes |
|----------|------|------------|-------|
| ISO 4028 | ISO | ✅ HIGH | Direct equivalent — dog point |
| DIN 915 | DIN | ✅ HIGH | Direct equivalent (withdrawn, replaced by ISO 4028) |
| IS 6094-3 | BIS | 🟡 MEDIUM | Indian set screw Part 3 |

**Dimension facts:**
- Thread range: M1.6 to M24
- Drive: hexagon socket
- Point: dog point (cylindrical pin protruding from end — can be full dog or half dog)
- Full dog or half dog determined by diameter and length:
  - M8, length ≤ 10mm → half dog (z = 2–2.25mm)
  - M8, length > 10mm → full dog (z = 4–4.3mm)
- Material: steel 45H; stainless A2, A4

**Point behaviour:** Dog point engages a machined hole or slot in the mating shaft,
providing positive axial and rotational locking. Highest holding power of all types.

**Original Summary:**
GOST 11075-93 specifies set screws with a dog (pin) point — a short cylindrical
protrusion that engages a corresponding machined depression or hole in the mating
shaft. This design provides both axial and rotational locking, making it the strongest
holding set screw configuration. The pin can be full-length or half-length depending
on the combination of screw diameter and length. Direct equivalents are ISO 4028 and
the withdrawn DIN 915. Applications include gear-to-shaft connections, pulley hubs,
cam assemblies, and any precision location where the screw must both lock and locate
a component on a shaft.

**Data sources consulted:** dinmark.com.ua, eurolinkfss.com, iso.org

### ISO 4027 / DIN 914 — Cone Point Set Screw
**Title:** Hexagon Socket Set Screw — Cone Point
**GOST status:** Specified as ISO 4027 in modern Russian engineering; legacy GOST number unconfirmed

**Cross-References:**
| Standard | Body | Notes |
|----------|------|-------|
| ISO 4027 | ISO | Cone point set screw |
| DIN 914 | DIN | Direct equivalent (withdrawn) |
| IS 6094-2 | BIS | Indian cone point set screw |

**Point behaviour:** Cone point concentrates force at a point. Suitable for permanent
installations into softer materials or pre-spotted holes. Provides highest axial
locking force but causes surface marking.

**Original Summary:**
The cone point set screw (ISO 4027 / DIN 914) uses a conical tip that either locates
into a machined dimple or penetrates softer shaft material to provide strong axial
holding. The deep penetration on tightening gives high axial and holding power; in
materials harder than Rockwell C15, the point should be spotted to half its length
to develop adequate shear strength. Used in permanent shaft-hub connections, pivot
applications, and fine adjustment mechanisms. In Russian engineering practice, this
type is typically called out as ISO 4027 directly; the Indian equivalent is IS 6094
Part 2.

---

### ISO 4029 / DIN 916 — Cup Point Set Screw
**Title:** Hexagon Socket Set Screw — Cup Point
**GOST status:** Most commonly used type globally; specified as ISO 4029 in Russian practice

**Cross-References:**
| Standard | Body | Notes |
|----------|------|-------|
| ISO 4029 | ISO | Cup point set screw — most common type worldwide |
| DIN 916 | DIN | Direct equivalent (withdrawn) |
| IS 6094-4 | BIS | Indian cup point set screw |

**Point behaviour:** Cup point (concave tip) bites into shaft surface when tightened.
Best general-purpose set screw. Works against most shaft materials. Suitable for
repeated removal and reinstallation.

**Original Summary:**
The cup point set screw (ISO 4029 / DIN 916) is the most widely used set screw type
globally. Its concave cup-shaped tip bites into the mating shaft surface upon tightening,
creating an interference fit that resists both axial movement and rotation. Unlike
cone or dog points, the cup point distributes its bite across a small ring contact
area rather than a point, reducing the risk of catastrophic surface damage. Best
suited for use against softer shaft materials (steel, aluminium, brass). Widely used
for securing pulleys, gears, and collars to shafts in general machinery. Indian
equivalent is IS 6094 Part 4.

**Data sources consulted:** eurolinkfss.com, aspenfasteners.com, fasteners.eu

## SECTION C: LOCK NUTS (PREVAILING TORQUE TYPE)

**Important concept:** Lock nuts resist loosening by creating prevailing torque —
friction that must be overcome in both tightening and loosening directions.
Two main types: (1) Nylon insert (non-metallic) and (2) All-metal (distorted thread).

---

### GOST 50592-93 / GOST 11872-89
**Title:** Hexagon Lock Nut with Nylon Insert (Nyloc Nut)
**Category:** Lock nut — non-metallic insert type

**Cross-References:**
| Standard | Body | Confidence | Notes |
|----------|------|------------|-------|
| ISO 7040 | ISO | ✅ HIGH | Nylon insert lock nut, style 1 (regular height) |
| ISO 7041 | ISO | ✅ HIGH | Nylon insert lock nut, high form |
| DIN 985 | DIN | ✅ HIGH | Low-form nyloc (→ ISO 7040) |
| DIN 982 | DIN | ✅ HIGH | High-form nyloc (→ ISO 7041) |
| IS 7002 | BIS | ✅ HIGH | Indian prevailing torque hex nut standard |

**⚠ KEY DISTINCTION:**
- DIN 985 / ISO 7040 = low-form (standard height nyloc) — most common
- DIN 982 / ISO 7041 = high-form (taller nut, more thread engagement)
- DIN 982 height > DIN 985 height for same diameter

**Dimension facts:**
- Thread range: M3 to M39 (ISO 7040)
- Property classes: 5, 6, 8, 10 (steel); A2, A4 (stainless)
- Nylon insert: polyamide (PA6 or PA66)
- Temperature limit: ~120°C continuous (nylon degrades above this)
- Thread tolerance: 6H
- One-way installation: must be installed top-up

**⚠ CRITICAL WARNING for users:**
Do NOT use nylon insert lock nuts above 120°C or in contact with:
- Strong acids or solvents (degrade nylon insert)
- Environments requiring sterilisation
For high-temperature or chemical applications → use all-metal lock nut (GOST/ISO 7042)

**Original Summary:**
GOST 50592-93 (and related GOST 11872-89) specifies prevailing torque lock nuts that
use a permanently deformed non-metallic (nylon/polyamide) insert to create friction
against the mating bolt thread, resisting loosening under vibration and dynamic loads.
The nylon insert is undersized relative to the bolt thread and does not cut into it,
allowing limited reuse. These are among the most widely used locking fasteners in
general machinery. Low-form variants correspond to DIN 985 / ISO 7040; high-form
variants to DIN 982 / ISO 7041. The Indian standard IS 7002 covers prevailing torque
hexagon nuts of this type. Key limitation: the nylon insert limits service temperature
to approximately 120°C; for higher temperatures, all-metal lock nuts (ISO 7042) must
be substituted.

**Data sources consulted:** aspenfasteners.com (DIN 982 spec), fasteners.eu, IS 7002:2005 PDF,
fastenersclearinghouse.com

---

### ISO 7042 / DIN 980 — All-Metal Prevailing Torque Lock Nut
**Title:** All-Metal Hexagon Prevailing Torque Lock Nut (Stover Nut)
**GOST equivalent:** Referenced as ISO 7042 in Russian practice; some legacy drawings use DIN 980

**Cross-References:**
| Standard | Body | Confidence | Notes |
|----------|------|------------|-------|
| ISO 7042 | ISO | ✅ HIGH | All-metal prevailing torque nut |
| DIN 980 | DIN | ✅ HIGH | German equivalent (Stover / oval lock nut) |
| IS 7002 | BIS | 🟡 MEDIUM | IS 7002 covers both insert and all-metal types |

**Also known as:** Stover nut, oval lock nut, crimped lock nut, top lock nut

**How it works:** The top of the nut is distorted (oval or crimped) so it clamps the
bolt thread mechanically. No nylon insert — purely metallic locking action.

**Dimension facts:**
- Thread range: M5 to M39
- Property classes: 5, 6, 8, 10, 12 (steel); A2, A4 (stainless)
- Temperature rated: up to 300°C+ (no polymer insert)
- One-way installation: top-up (distorted end goes last)
- Reusability: limited — distorted section deforms further with each use

**Advantage over nyloc:** Suitable for high temperatures, chemical environments,
and applications where nylon degradation is a concern.

**Original Summary:**
The all-metal prevailing torque lock nut (ISO 7042 / DIN 980) achieves its locking
action through controlled distortion of the top portion of the nut body rather than
a polymer insert. When tightened, the distorted threads bite mechanically into the
mating bolt, creating resistance to loosening without any non-metallic component.
This makes it suitable for elevated temperatures (up to 300°C or higher depending on
material grade), aggressive chemical environments, and applications where nylon insert
nuts are prohibited. Also known as a Stover nut. The Indian standard IS 7002 covers
this type alongside nylon insert nuts. Commonly used in exhaust systems, engines,
and industrial equipment operating above the temperature limit of nylon-insert nuts.

**Data sources consulted:** aspenfasteners.com (DIN 980 spec), fastenersclearinghouse.com,
fasteners.eu, ISO 7042 scope

## SECTION D: MARKING REQUIREMENTS (CATEGORY 2)

**Governing standards:** ISO 898-1 Cl. 11, ISO 898-5 Cl. 8, ISO 2320 Cl. 7

### Socket Head Cap Screws (GOST 11738-84 / ISO 4762)

Marking is on the top bearing face of the cylindrical head (around the socket opening) or
on the cylindrical side face. The marking location choice depends on the head diameter:
smaller sizes use the top face; larger sizes may use both locations.

| Property Class | Marking location | Method |
|----------------|-----------------|--------|
| 8.8 | Top face or side | Embossed or indented |
| 10.9 | Top face or side | Embossed or indented |
| 12.9 | Side face only | Embossed (raised) — **never indented on top face** |
| A2-70 / A4-70 | Top face or side | Embossed |

**⚠ CLASS 12.9 RULE:** Indentation marking on the top face of a class 12.9 socket head screw creates a stress concentration directly adjacent to the highest-stress region (socket lip). Only embossed or side-face marking is permitted. This is an explicit requirement of ISO 898-1 Annex B.

Manufacturer identification code is mandatory for class 8.8 and above, placed adjacent to the property class mark.

### Set Screws (ISO 898-5 / GOST 11074/11075 series)

Set screws have minimal marking area due to their headless, compact form. Marking practice:
- On the hex socket end face: class (e.g., "45H") for M6 and above where space permits
- For M3 and below: marking is impractical; set screws are supplied in labelled packaging with full traceability through lot marking on the packaging, not the fastener

**Set screw hardness class marking (ISO 898-5):**

| Class | Min Hardness | Max Hardness | Typical use |
|-------|-------------|-------------|-------------|
| 14H | 140 HV | 400 HV | Mild steel, non-hardened |
| 22H | 220 HV | 400 HV | General use, moderate load |
| 33H | 330 HV | 420 HV | Hardened shaft applications |
| 45H | 450 HV | 560 HV | Standard commercial supply; hardened alloy steel |

Most commercially available set screws in sizes M3–M24 are class 45H (fully hardened alloy steel). Stainless set screws are marked A1, A2, or A4 on the socket end.

### Lock Nuts — Nylon Insert (GOST 50592-93 / ISO 7040 / DIN 985)

Marked on the flat face of one hexagon:
- Property class: "5", "6", "8", "10"
- Stainless: "A2" or "A4" with strength suffix where applicable
- Nylon insert colour is manufacturer-specific and NOT standardised for property class identification — do not rely on insert colour alone to identify the property class

### Lock Nuts — All-Metal (ISO 7042 / DIN 980)

Same marking location and format as standard hex nuts. The distorted top section of the nut is a visual identifier of type but does not replace the property class marking on the flat.

## SECTION E: SURFACE TREATMENTS (CATEGORY 2)

Coating code table (GOST 9.301 system) is the same as Category 1, Section 9. The following notes apply specifically to Category 2 fastener types.

### Socket Head Cap Screws

| Coating | Use case | Notes |
|---------|----------|-------|
| Black oxide + oil (code 05/special) | General engineering, class 12.9 | Low-cost; mild indoor corrosion resistance; no hydrogen embrittlement risk |
| Zinc electroplating (code 01, 6 µm) | General use, class 8.8 and 10.9 | Most common; post-plate baking mandatory for class 10.9 |
| Mechanical zinc plating | Class 10.9 and 12.9 | No acid bath; eliminates hydrogen embrittlement risk |
| Stainless A2 / A4 | Corrosive, food-grade, pharmaceutical | Lower torque capacity; anti-seize essential |
| Geomet / Dacromet (zinc flake) | Automotive, marine, class 12.9 | Superior corrosion resistance; no hydrogen embrittlement |

**⚠ SOCKET FIT AFTER PLATING:** Zinc electroplating deposits coating inside the hex socket drive. Plating thickness of 6 µm reduces the effective socket diameter and can cause hex key binding. Pre-plating socket dimensions must account for coating build-up. Always verify hex key fit on a plated sample before bulk acceptance.

### Set Screws

- Supplied predominantly as black oxide (code 40 / bare) since the screw body is concealed inside the threaded hole
- Stainless set screws require no additional coating
- For food, pharmaceutical, or aggressive chemical environments: A4 (316) stainless is the standard specification
- Hot-dip galvanizing is impractical for set screws due to thread tolerance impact at small diameters

### Lock Nuts — Nylon Insert Type

**Hot-dip galvanizing is prohibited for nylon insert lock nuts.** The zinc bath operates at approximately 450°C — well above the 120°C degradation limit of polyamide (PA6/PA66). The nylon insert will melt and destroy the locking mechanism.

Acceptable coatings for nylon insert lock nuts:
- Zinc electroplating (cold process, ≤ 60°C bath): acceptable
- Mechanical zinc plating: acceptable
- Geomet / Dacromet (ambient temperature cure): acceptable
- Stainless A2 / A4 body (no additional coating): standard for corrosive environments

### Lock Nuts — All-Metal Type (ISO 7042 / DIN 980)

All surface treatments are compatible including hot-dip galvanizing. For elevated temperature service (>200°C), bare steel (no coating) or phosphate + oil is preferred as some coating systems may degrade at sustained high temperatures.

## SECTION F: MATERIAL GRADE SPECIFICATIONS (CATEGORY 2)

**Governing standards:** ISO 898-1, ISO 898-5, ISO 3506, GOST 1759.4-87

### Socket Head Cap Screws (ISO 4762 / GOST 11738-84)

| Property Class | Steel type | Typical GOST steel grade | Heat treatment |
|----------------|-----------|--------------------------|---------------|
| 8.8 | Medium carbon or Cr-alloy | Steel 35, Steel 40X | Quenched and tempered |
| 10.9 | Cr-alloy | Steel 40X, 30HGSA | Quenched and tempered |
| 12.9 | Cr or Ni-Cr-Mo alloy; controlled chemistry | 35HGSA, 40HN2MA | Quenched and tempered; strict S/P limits |
| A2-70 | Austenitic 18Cr-8Ni (304) | 12X18H9 | Cold worked to 700 MPa |
| A4-70 | Austenitic 17Cr-12Ni-2Mo (316) | 10X17H13M2T | Cold worked to 700 MPa |

Class 12.9 material specification per ISO 898-1 Annex A: sulphur ≤ 0.035%, phosphorus ≤ 0.035%, no conventional acid-bath electroplating. Carbon equivalent is strictly controlled to prevent delayed fracture under hydrogen exposure.

### Set Screws (ISO 898-5 / GOST 11074/11075)

| Class | Steel | Typical GOST grade | Min hardness |
|-------|-------|--------------------|-------------|
| 14H | Low carbon | Steel 10, 20 | 140 HV |
| 22H | Medium carbon | Steel 35 | 220 HV |
| 33H | Medium carbon, hardened | Steel 40, 40X | 330 HV |
| 45H | Alloy steel, fully hardened | Steel 40X, 38XA | 450 HV |

**Important:** The set screw point must be at least as hard as the shaft material it contacts. For case-hardened shafts (50+ HRC), a class 45H set screw with cone or dog point is required. Flat-point set screws on very hard surfaces may slip under load regardless of tightening torque.

### Lock Nuts (ISO 898-2, ISO 2320, GOST 1759.5-87)

| Type | Property class | Steel | Nylon insert material |
|------|---------------|-------|----------------------|
| Nylon insert — low form (ISO 7040) | 5, 6, 8, 10 | Medium carbon | PA6 or PA66 (polyamide) |
| Nylon insert — high form (ISO 7041) | 5, 6, 8, 10 | Medium carbon | PA6 or PA66 |
| All-metal (ISO 7042) | 5, 6, 8, 10, 12 | Carbon or alloy, top distorted | — |
| Stainless nylon insert | — | A2 or A4 | PA6 insert |

PA66 (nylon 66) has a slightly higher heat resistance (~130°C continuous) than PA6 (~120°C continuous), but both are below the threshold for hot processing environments. Verify the specific insert material in the product data sheet when operating near the limit.

## SECTION I: INSTALLATION TORQUE REFERENCE (CATEGORY 2)

**Basis:** Indicative values for 90% proof load utilisation, µ = 0.14 (zinc-plated, clean, dry).
See Category 1 Section 13 for friction adjustment factors.

### Socket Head Cap Screws — Tightening Torque (Metric Coarse, Dry)

| Size | Pitch (mm) | Class 8.8 (Nm) | Class 10.9 (Nm) | Class 12.9 (Nm) |
|------|-----------|---------------|----------------|----------------|
| M3 | 0.5 | 1.3 | 1.8 | 2.2 |
| M4 | 0.7 | 3.0 | 4.2 | 5.1 |
| M5 | 0.8 | 6.0 | 8.5 | 10 |
| M6 | 1.0 | 10 | 14 | 17 |
| M8 | 1.25 | 25 | 35 | 41 |
| M10 | 1.5 | 49 | 69 | 83 |
| M12 | 1.75 | 84 | 120 | 144 |
| M16 | 2.0 | 210 | 295 | 355 |
| M20 | 2.5 | 410 | 580 | 690 |
| M24 | 3.0 | 710 | 1000 | 1200 |

**Stainless A2-70 / A4-70:** Apply anti-seize (Molykote G-Rapid or equivalent) and use approximately 65% of the class 8.8 values. Without anti-seize, stainless-to-stainless contact will gall (cold-weld) before the desired torque is reached.

**Anti-galling precautions for stainless socket head screws:**
1. Apply anti-seize to bolt threads and socket bore before assembly
2. Use a calibrated torque wrench — do not impact-drive stainless socket head screws
3. Tighten slowly (< 1 revolution per second) to allow lubrication to distribute
4. If resistance increases suddenly during tightening, stop and back out — galling has likely started

### Set Screw Installation Torque (Class 45H, Coarse Thread, Dry)

These are the torques to apply to the socket drive for achieving the required axial clamping force against the shaft.

| Size | Pitch (mm) | Installation torque (Nm) |
|------|-----------|--------------------------|
| M3 | 0.5 | 0.7 |
| M4 | 0.7 | 1.5 |
| M5 | 0.8 | 3.0 |
| M6 | 1.0 | 5.5 |
| M8 | 1.25 | 13 |
| M10 | 1.5 | 25 |
| M12 | 1.75 | 44 |
| M16 | 2.0 | 108 |

**Point-type effect on clamping:** Dog point and cone point set screws achieve higher axial holding force at the same installation torque compared to flat and cup point, because their geometry concentrates force rather than distributing it. For maximum holding power with a cup point, a shallow dimple (spotted hole, half the screw diameter deep) in the shaft is recommended.

### Lock Nut Installation Torque

Tighten nylon insert and all-metal lock nuts to the same torque as the mating bolt specification. The prevailing torque friction is inherent in the nut design and adds automatically to the assembly — no additional torque beyond the standard bolt torque is required or recommended. Over-torquing beyond the bolt's proof load does not improve locking performance.

## SECTION J: REUSABILITY GUIDELINES (CATEGORY 2)

### Socket Head Cap Screws

Follow the same rules as hex bolts (Category 1, Section 14). Additional checks specific to socket head cap screws:

- **Socket condition:** Insert the correct-size hex key — it must fit with minimal play. Excessive clearance (socket walls rounded by previous cam-out) indicates over-torque or wrong key size was used. Replace the screw.
- **Head-to-shank fillet:** Inspect the underside fillet under magnification if the screw was overtorqued or shock-loaded. Cracks initiate here in class 10.9 and 12.9 screws.
- **12.9 screws after overload:** Replace without exception. Class 12.9 fasteners subjected to proof load exceedance are at risk of delayed hydrogen-assisted fracture.

Recommended maximum reuse cycles mirror Category 1, Section 14 recommendations (5 cycles for class 8.8; 3 cycles for class 10.9; 2 cycles for class 12.9 under conservative practice).

### Set Screws

| Point type | Reusability |
|-----------|-------------|
| Flat point | Multiple reuse — provided point face shows no mushrooming |
| Cup point | Multiple reuse — inspect point rim for deformation; replace if rim is collapsed |
| Dog point | Fully reusable — inspect pin for wear and socket for rounding |
| Cone point (into a dimple) | Multiple reuse in the same dimple — the cone relocates into the hardened dimple without further deformation |
| Cone point (into undimpled shaft) | Single use at that shaft location — the cone permanently deforms the shaft surface |

### Nylon Insert Lock Nuts (ISO 7040 / DIN 985 / GOST 50592-93)

**Maximum 5 assembly cycles** per ISO 2320. This limit is not a recommendation — it is the test basis for the standard. The prevailing torque is measured on the 5th on-cycle during type approval testing. Beyond 5 cycles, the insert is no longer guaranteed to meet minimum torque.

Mandatory discard conditions regardless of cycle count:
- Visible nylon damage (cracking, deformation beyond thread profile)
- Measured prevailing torque falls below ISO 2320 minimum for the size and class
- Any heat exposure above 120°C (polyamide degrades; insert will not function reliably on reuse)
- Any chemical exposure to strong solvents, concentrated acids, or hydraulic oil at elevated temperature

### All-Metal Prevailing Torque Lock Nuts (ISO 7042 / DIN 980)

**Maximum 3–5 assembly cycles** depending on material grade. Larger sizes (M20+) with higher prevailing torque loads degrade faster and should be treated as 3-cycle maximum. Measure prevailing torque after each removal; compare against ISO 2320 minimum values. Replace when measured torque falls below specification.
