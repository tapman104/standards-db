# Standard Overview

IS 2269 specifies socket head cap screws (Allen bolts) for metric threads. Harmonised with ISO 4762. Direct GOST equivalent: GOST 11738-84. The DIN 912 predecessor is withdrawn; ISO 4762 is the current international reference.
