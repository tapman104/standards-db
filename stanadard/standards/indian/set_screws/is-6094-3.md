# Standard Overview

IS 6094 Part 3 specifies socket set screws with dog point (full or half dog). Equivalent to ISO 4028. Direct GOST equivalent: GOST 11075-93. DIN 915 is the withdrawn German predecessor. Requires machined shaft recess.
