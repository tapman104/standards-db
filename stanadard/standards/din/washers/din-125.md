# Standard Overview

DIN 125 was the German standard for plain washers, published in two parts:
- **DIN 125-A**: No chamfer on bore → replaced by ISO 7089 (200HV)
- **DIN 125-B**: Chamfered bore → replaced by ISO 7090 (300HV)

**Both withdrawn.** Replacements are ISO 7089 (Type A) and ISO 7090 (Type B). Dimensions identical.

# Withdrawal Note

DIN 125-A and DIN 125-B are **withdrawn**. Specify ISO 7089 or ISO 7090 for new orders.
