# Standard Overview

DIN 912 was the German standard for hexagon socket head cap screws (Allen bolts). **Withdrawn** and replaced by ISO 4762. Dimensions are essentially identical; substitution is safe at all common sizes. DIN 912 remains one of the most widely referenced legacy part numbers globally despite withdrawal.

# Withdrawal Note

DIN 912 is **withdrawn**. Specify ISO 4762 for new designs and replacement orders. DIN 912 continues to appear on legacy drawings; supply chain reference is safe through ISO 4762.
