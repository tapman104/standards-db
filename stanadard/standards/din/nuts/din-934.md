# Standard Overview

DIN 934 was the German standard for hexagon nuts, style 1 (standard height). **Withdrawn** and replaced by ISO 4032. Dimensions are identical to ISO 4032; substitution is safe for all sizes. Minor tolerance grade differences possible at smaller sizes (M1–M3).

# Withdrawal Note

DIN 934 is **withdrawn**. Specify ISO 4032 for new orders. Safe substitution at all common sizes.
