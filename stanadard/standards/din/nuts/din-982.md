# Standard Overview

DIN 982 was the German standard for nylon insert lock nuts (nyloc), high form. **Withdrawn** and replaced by ISO 7041. Dimensions identical to ISO 7041; substitution is safe.

# Withdrawal Note

DIN 982 is **withdrawn**. Specify ISO 7041 for new orders.
