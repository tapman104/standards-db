# Standard Overview

DIN 985 was the German standard for nylon insert lock nuts (nyloc), low form. **Withdrawn** and replaced by ISO 7040. Dimensions identical; temperature and reuse limits unchanged. HDG prohibited.

# Withdrawal Note

DIN 985 is **withdrawn**. Specify ISO 7040 for new orders.
