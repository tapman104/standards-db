# Standard Overview

ISO 4026 specifies socket set screws with flat point (no protrusion at tip). Used to lock shaft-to-hub connections, adjust valve seats, and secure components where minimal shaft marring is required. The flat tip bears on the shaft surface — lowest holding force of all set screw point types but least surface damage.

Direct GOST equivalent: GOST 11074-93. German predecessor: DIN 913 (withdrawn, replaced by ISO 4026). Indian: IS 6094-1.
