# Standard Overview

ISO 4016 specifies grade C (black/rough) hexagon head bolts with partial thread shank. Grade C has wider dimensional tolerances than grades A and B, making it suitable for general structural, construction, and agricultural applications where surface finish and tolerance precision are not critical. Maximum property class is 5.8 — these bolts are not produced in class 8.8+.

Direct GOST equivalent: GOST 15589-70. Indian equivalent: IS 1363-1. DIN 601 is the withdrawn German predecessor.

# Mechanical Properties

Grade C: property classes 4.6, 5.6, 5.8 only. **Not available in 8.8 or higher.**

# Surface Treatments

Common: hot-dip galvanized (HDG) per ISO 10684 — used extensively in structural/civil applications. Black (as-rolled) also common.

# Reliability Notes

- Grade C only — not a substitute for grade A or B bolts
- Property class limited to 5.8 maximum
- DIN 601 withdrawn; ISO 4016 is current standard
