# Standard Overview

ISO 7380 specifies socket button head screws — a low-profile, dome-headed variant of the socket cap screw. The wide, rounded head distributes load over a large bearing area and provides an attractive, low-snag profile. Used in consumer electronics, furniture, sports equipment, and machinery covers where head height clearance is limited.

**No GOST equivalent exists** — Russian and Soviet technical drawings specify ISO 7380 directly. No DIN or IS equivalent. This is an ISO-only standard.

# Limitations vs. ISO 4762

The button head has lower socket depth than ISO 4762, resulting in lower maximum tightening torque capacity. Do not substitute ISO 7380 for ISO 4762 in high-clamp-load applications.

# Reliability Notes

- No national standard equivalent — always specify ISO 7380 explicitly
- Maximum torque lower than ISO 4762 at same thread size — do not over-torque
- Tamper-evident/anti-tamper versions available but not in ISO 7380 scope
