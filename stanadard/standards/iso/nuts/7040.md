# Standard Overview

ISO 7040 specifies the low-form nylon insert lock nut (commonly called nyloc nut). The polyamide insert provides a prevailing torque that resists vibration-induced loosening without requiring adhesive or secondary locking features. Specified per ISO 2320 prevailing torque test protocol.

Direct GOST equivalent: GOST 50592-93. German predecessor: DIN 985 (withdrawn, replaced by ISO 7040). Indian: IS 7002 (covers both low and high form).

# Reusability

Maximum 5 cycles. Discard if prevailing torque falls below ISO 2320 minimum on any cycle. See standards/gost/lock_nuts/50592-93.md for full reuse protocol.
