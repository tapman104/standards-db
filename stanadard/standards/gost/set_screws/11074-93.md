# Standard Overview

GOST 11074-93 specifies headless set screws with a flat (blunt) end and an internal hexagon socket drive. These screws are threaded along their entire length and are used to secure machine components where conventional bolt-and-nut assembly is impractical — primarily for locking hubs, collars, gears, and mechanical stops onto shafts.

The flat point minimises surface damage on the mating component and provides the largest contact area of all set screw point types, making it suitable for locking parts against hardened shafts or precision surfaces. It corresponds directly to ISO 4026 and the withdrawn DIN 913. Applications include instrument housings, encoder rings, mechanical stops, and adjustment mechanisms in precision machinery.

Most commercially available set screws M3–M24 are class 45H. Stainless: A2 or A4.

**Point behaviour:** Flat point distributes load over the largest area of all set screw types. Recommended for:
- Use against hardened surfaces (surface marking minimised)
- Precision instruments where shaft surface must be preserved
- Adjustment mechanisms requiring frequent repositioning

# Surface Treatments

Set screws are predominantly supplied black oxide (phosphate + oil) or bare, since the screw body is concealed inside the threaded hole. Hot-dip galvanizing is impractical due to thread tolerance impact at small diameters. Stainless requires no additional coating.

# Marking Requirements

- Hardness class on the socket end face (M6 and above where space permits)
- M3 and below: marking impractical — traceability via packaging lot label only
- Stainless: "A2" or "A4" on socket end where size allows

# Reusability

Flat point set screws are fully reusable. Inspect the flat point face for mushrooming or deformation. Replace if point face is no longer flat.

# Reliability Notes

- HIGH confidence: GOST 11074-93 = ISO 4026 = DIN 913 confirmed (dinmark.com.ua + multiple)
