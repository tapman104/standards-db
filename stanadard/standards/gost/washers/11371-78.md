# Standard Overview

GOST 11371-78 specifies flat circular washers used to distribute clamping load under bolt heads or nuts, protecting the mating surface from damage and bridging oversized clearance holes. Two versions are defined: Version 1 without outer chamfer (equivalent to ISO 7089 / DIN 125A) and Version 2 with chamfer on the outer edge (equivalent to ISO 7090 / DIN 125B). Accuracy class A offers tighter dimensional control for precision applications; class C is suitable for general construction.

Coverage spans M1 through M48, available in carbon steel, stainless steel (A2, A4), and aluminium alloys (AD1, AMg) for specialised pneumatic and hydraulic applications. Hardness selection must match the bolt property class.

# Dimensions

| Parameter | Value |
|-----------|-------|
| Thread range | M1 to M48 |
| Versions | 1 (no chamfer) and 2 (chamfer on OD) |
| Accuracy classes | A (precision) and C (general) |
| Thickness tolerance | Tighter for class A than class C |

Washer inside diameter is specified as slightly larger than nominal bolt diameter to allow easy assembly.

# Mechanical Properties

| Hardness | Application |
|----------|-------------|
| 200 HV | Standard — for bolts property class ≤ 8.8 |
| 300 HV | Required for bolts property class 10.9 and 12.9 (prevents washer embedment under high clamp load) |

**⚠ Important:** Always specify 300 HV washers when used with class 10.9 or 12.9 bolts. A soft washer will embed into the mating surface under high clamp load, reducing bolt preload over time.

# Marking Requirements

Hardness class (200 HV or 300 HV) is specified in the designation string. Individual washers are typically not marked; traceability through lot labelling and delivery documentation.

# Testing Requirements

| Test | Criterion |
|------|-----------|
| Hardness | Within 200 HV or 300 HV range per specification |
| Dimensional | OD, ID, thickness within tolerance field per class A or C |
| Flatness | Per GOST 11371-78 table |
| Chamfer (Version 2) | Chamfer dimensions within tolerance |

# Designation Examples

**GOST format:**
```
Washer 12 — 200 HV GOST 11371-78
  (Washer for M12 bolt; 200 HV; Version 1 no chamfer)

Washer 16 — 300 HV GOST 11371-78
  (Washer for M16 bolt; 300 HV; for use with class 10.9/12.9 bolts)

Washer 10 — Version 2 — 200 HV GOST 11371-78
  (Washer for M10 bolt; chamfered outer edge; 200 HV)
```

**ISO equivalent:**
```
ISO 7089 — 12 — 200 HV      (Version 1, no chamfer)
ISO 7090 — 12 — 200 HV      (Version 2, chamfer)
```

**IS equivalent:**
```
Washer IS 2016 — 12 mm, Grade 1
```

# Reusability

Reusable indefinitely provided bearing faces show no embedment damage, distortion, or corrosion affecting load distribution.

# Reliability Notes

- HIGH confidence: GOST 11371-78 = ISO 7089/7090 confirmed across multiple sources
- DIN 125A/B superseded by ISO 7089/7090 — use ISO designation in new designs
- Hardness selection is critical — 200 HV washers must NOT be used with class 10.9/12.9 bolts
