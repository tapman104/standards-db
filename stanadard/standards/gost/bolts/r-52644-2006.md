# Standard Overview

GOST R 52644-2006 provides specific dimensional and mechanical requirements for high-strength hexagon bolts with enlarged (large) width-across-flats dimensions, intended for structural steel bolting applications. The enlarged head ensures adequate torque application during controlled tightening and provides sufficient bearing area on structural steel surfaces. Property class is 10.9 minimum.

This standard is the dimensional companion to GOST R 52643-2006 (general assembly requirements) and maps directly to ISO 7411 and DIN 6914. It represents the modern Russian Federation update to the earlier GOST 22353-77 for structural bolt dimensions.

# Dimensions

| Parameter | Value |
|-----------|-------|
| Thread range | M12 to M36 |
| Head type | Hexagon, large WAF |
| Product grade | C (structural) |
| WAF | Enlarged vs standard ISO 4014 hex bolt for same diameter |

Large WAF provides:
- Higher torque capacity without head rotation
- Larger bearing area on structural steel
- Easier installation with standard structural wrenches

# Mechanical Properties

Same as GOST R 52643-2006. Min property class 10.9:
- Min tensile: 1000 MPa
- Min yield: 900 MPa
- Hardness: 310–372 HV

# Materials

Alloy steel, quenched and tempered. Typical GOST grades: 40X, 40HN. See GOST R 52644-2006 Table for exact material specifications.

# Surface Treatments

Same as GOST 22353-77 (structural bolt). Hot-dip galvanizing standard for exposed structural steel.

# Marking Requirements

Same as GOST R 52643-2006. Property class "10.9" + manufacturer code + GOST R 52644 identification.

# Testing Requirements

Same test matrix as GOST 22353-77. Assembly pretension test per EN 14399-2 / GOST R 52643.

# Reliability Notes

- HIGH confidence: ISO 7411 / DIN 6914 / EN 14399-4 confirmed
- Use together with GOST R 52643-2006 as the system specification
- Single-use rule applies — pretensioned structural bolts must not be reused
