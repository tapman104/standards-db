# Standard Overview

GOST 5915-70 defines the standard hexagon nut (style 1) with normal height, produced to accuracy class B. It is the most commonly paired nut for use with GOST 7798-70 (partial thread hex bolt) and GOST 7805-70 (full thread hex bolt) in general mechanical and structural applications. Style 1 nuts have a height approximately 0.8 times the nominal thread diameter (sometimes called "normal" height).

The standard is dimensionally equivalent to ISO 4032 and the now-withdrawn DIN 934, both of which share the same width-across-flats and height dimensions. The Indian standard IS 1364 Part 3 is declared identical to ISO 4032 by BIS. Property classes from 5 through 12 are available.

Governed by GOST 1759.5-87 (equivalent to ISO 898-2).

**Nut-bolt pairing rules:**
- Class 5 nut → pairs with class 4.6 or 5.6 bolt
- Class 8 nut → pairs with class 8.8 bolt
- Class 10 nut → pairs with class 10.9 bolt
- Class 12 nut → pairs with class 12.9 bolt

# Surface Treatments

Same coating code table as bolts (GOST 9.301). Most common:
- Code 01: Zinc electroplating 6 µm — general use
- Code 13: Hot-dip galvanizing — structural/outdoor

**HDG nuts:** Tapped oversize after galvanizing to fit HDG bolts. Must be ordered and matched with HDG bolts from same coating batch.

# Marking Requirements

Marked on one hexagon side flat:
- Property class number (e.g., "5", "8", "10", "12")
- Manufacturer code for class 8, 10, 12
- Stainless: "A2" or "A4"
